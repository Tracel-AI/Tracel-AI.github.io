This script is run before a PR is created.

It is used to check that the code compiles and passes all tests.

It is also used to check that the code is formatted correctly and passes clippy.

To build this script, run the following command:

rustc scripts/run-checks.rs â€“crate-type bin â€“out-dir scripts

To run the script:

./scripts/run-checks environment

where environment can assume ONLY the following values: - std to perform checks using libstd - no_std to perform checks on an embedded environment using libcore - typos to check typos in the source code - examples to check the examples compile
